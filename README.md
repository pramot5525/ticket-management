# Ticket Management

## Local Setup

1. Run Backend Express and MySQL server

Run docker compose in _root_ directory

```sh
docker-compose up
```

When it is running, Backend open port is :3000 and mysql open port is :3306.

2. Run Frontend Vue.js

```sh
cd client
yarn install
yarn serve
```

Open web browser to [localhost:8080](http://localhost:8080)
